Brief instruction to build the app:

1. Using GridLayout to build the calculator display and keyboard. And there are total 16 buttons, and one textDisplay.

2. There are four main types of clickListners which are NumericOnClickListener, SignOnClickListner, ClearOnClickListner, EqualClickListener. Each type of clicklistner represents one independent button logic. 

3. There are three boolean flag which handle situations which are previous sign click, error state and first input. 

4. The calculator can proper handling error, rounding. And showing ERROR! / OVERFLOW! message on display in corespongding error situations.
